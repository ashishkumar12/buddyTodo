# buddyTodo

# Steps To Install

1. Clone this respository in your PC
2. Run (npm install) and (npm run sass)
3. To start the application run (npm start)

# Requirements

1. MongoDB
2. Node.js v8.x.x(LTS)
3. Grunt.js (to install run npm run grunt -g)
